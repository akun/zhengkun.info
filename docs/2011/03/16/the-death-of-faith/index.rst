戊wù：天干的第五位。

戌xū：地支的第十一位。

戍shù：会意。从人持戈。甲骨文字形,象人负戈守卫边疆。本义:防守边疆。

戎róng：会意。从戈,从十。“戈”是兵器,“十”是铠甲的“甲”。本义:古代兵器的总称。
